# Simplified Kubernetes Development with a Microservice Templating Framework

## Description
This final career work focuses on the production of a templating framework to simplify the adoption of microservice architectures in Kubernetes environments. The
framework will provide a local console line interface (CLI) tool for generating templates from configuration files and a server tool that offers an endpoint for uploading
and managing these templates. By introducing standardized templates and consistent practices, the framework reduces the number of professionals required to set up
and maintain microservices while making systems easier to manage and scale. In
the absence of such standardization, distributed systems often rely on inconsistently
structured manifests, which complicates operations and significantly increases the
effort required for ongoing management.
